
Focus on high-impact topics first:

1. **Primary Care Essentials** (most common conditions)
   - Hypertension, diabetes, hyperlipidemia
   - Depression, anxiety
   - Preventive care

2. **High-Volume Specialties**
   - Cardiovascular (heart disease is #1 cause of death)
   - Oncology (cancer screening and treatment)
   - Infectious disease (COVID-19, sepsis, UTIs)

3. **Evidence-Rich Topics**
   - Topics with many RCTs and systematic reviews
   - Use `--high-evidence` flag to filter

### Strategy 3: Batch Processing
Process topics in batches by specialty:

```bash
# Cardiovascular batch
npm run ingest:scale -- --topics-file cardiovascular-topics.txt --workers 5

# Metabolic batch
npm run ingest:scale -- --topics-file metabolic-topics.txt --workers 5
```

### Strategy 4: Use Checkpoint System
The checkpoint system allows you to:
- Resume from any point
- Skip completed topics
- Retry failed topics
- Track progress

```bash
# Resume from checkpoint
npm run ingest:scale -- --resume --workers 5
```

### Strategy 5: Optimize Settings
For comprehensive coverage:

```bash
# High-quality, evidence-based ingestion
npm run ingest:scale \
  --high-evidence \
  --from-year 2015 \
  --max-articles 500 \
  --workers 5 \
  --ncbi-key YOUR_KEY
```

### Strategy 6: Monitor and Adjust
1. Check checkpoint file regularly
2. Identify patterns in failures
3. Adjust batch sizes and parallelism
4. Use NCBI API key for higher rate limits
