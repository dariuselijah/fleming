# PubMed Day-1 Shards

- Workers per shard: 2
- Max per topic: 2000

## Run

Use one command per terminal/session:

```bash
bash data/eval/pubmed_shards/day1/launch_parallel.sh 12
```

Or copy individual lines from `commands.sh` and launch manually across terminals.
